# Betting Strategy (Thai Market)

## 1. โครงการนำเสนอให้คนไทย

1. **เริ่มด้วยมุมมอง** — ต่อ/รอง/สูง/ต่ำ/สเต็ป
2. **อ้างอิงราคา** — ระบุราคาต่อ (เช่น -0.5@0.92) หรือค่าน้ำไทย (-0.92)
3. **เหตุผล 2-3 ข้อ** — ฟอร์ม, แท็คติก, ตัวเจ็บ, โปรแกรม
4. **จบด้วยคำเตือน** — "เล่นพอสนุก" / "แบ่งไม้" / "รอราคาไลฟ์"

## 2. สูตรคำนวณเร็ว

- **Implied Probability** = 1 / decimal odds → ใช้ใน `scripts/calculate_odds.py`
- **Fair Line** = (1 / Σprob) * prob_i → ใช้ตัดสินว่าราคาปัจจุบันคุ้มไหม
- **Kelly Fraction (แบบง่าย)** = (bp - q) / b; แนะนำใช้ 25-50% Kelly สำหรับตลาดไทย

## 3. สไตล์การเดิมพันของนักเล่นไทย

| สไตล์ | ลักษณะ | วิธีตอบสนอง |
| --- | --- | --- |
| บอลเดี่ยวต่อแพง | ชอบทีมใหญ่ -1.25/-1.5 | เน้นเตือนเรื่องเกมรุกไม่คม / โปรแกรมถี่ |
| บอลรองเชียร์มันส์ | เล่นรองทีมเหนียว | ให้ข้อมูล H2H + ฟอร์มรับ |
| สเต็ป 3-5 ทีม | อยากคูณสูง | จัดลิสต์คู่เล็ก/สูงต่ำให้เลือก พร้อมความมั่นใจ ★ |
| ไลฟ์โอเวอร์ | ดูเกมก่อนแล้วกดสูง | ให้จุดสังเกตจังหวะ (เช่น TI>25 นาทีที่ 15) |

## 4. แนวคิดเพิ่มความน่าเชื่อถือ

- **Win Rate Tracking** — ใช้ `winrate_tracker.py` เก็บผล แสดง % รายเดือน
- **Confidence Stars** — ★★★ = มั่นใจสูง (form+ราคา align), ★★ = กลาง, ★ = เสี่ยง
- **Bankroll Note** — ใส่คำว่า "ลงทุนไม่เกิน 2-3% ของทุน" เพื่อดูมืออาชีพ

## 5. Workflow แนะนำ

1. ดึงราคาล่าสุดจาก odds API หรือหน้าระบบ UFAX9
2. คำนวณ implied probability → หา value gap เทียบโมเดล (เช่น projection จาก `tournament_projection.py`)
3. ผูกกับบริบทไทย (ข่าว, สนาม, เวลาแข่ง)
4. สรุปเป็น Thai paragraph + ปุ่ม CTA ฝาก/ถอน

## 6. ศัพท์ค่าน้ำไทยที่ควรใช้

- **น้ำแดง** = ค่าน้ำติดลบ เช่น -0.90 → จ่ายตามทุนถ้าเสีย
- **น้ำดำ** = ตัวเลขบวก เช่น 0.85 → เสียเต็ม ชนะได้ 0.85 เท่า
- **กินครึ่ง/เสียครึ่ง** = แฮนดิแคป .25/.75
- **ไหลต่อ/ไหลรอง** = ราคาขยับไปทางทีมต่อ/ทีมรอง
