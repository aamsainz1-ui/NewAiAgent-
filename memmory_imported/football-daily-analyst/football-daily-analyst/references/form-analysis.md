# Form & Momentum Analysis (Thai Football Context)

> ใช้คู่กับข้อมูลสดจาก football-data.org / TheSportsDB แล้วสรุปฟอร์มเป็นภาษาไทยแบบนักวิเคราะห์

## 1. ตัวชี้วัดหลัก 5 นัดหลังสุด

| Metric | วิธีดู | เกณฑ์ตีความ (ไทย) |
| --- | --- | --- |
| xG Differential | xG ได้ - xG เสีย ต่อเกม | > +0.5 = "บุกดุดัน", -0.2 ถึง +0.2 = "สูสี", < -0.5 = "เกมรับรั่ว" |
| Shots on Target Ratio | SOT ทีม / SOT รวม | >60% = "คุมเกมรุก", <40% = "ถูกกด" |
| Possession Territory | เวลาครองบอลฝั่งตรงข้าม (%) | >35% = "กล้าบีบสูง", <20% = "ถอยลึก" |
| Rest Differential | วันที่พักมากกว่า/น้อยกว่า | +2 วันขึ้นไป = ได้เปรียบสภาพร่างกาย |
| Travel Load | ระยะทางบิน (km) / จำนวนประเทศใน 10 วัน | >2 ประเทศ = เสี่ยงล้า (สำคัญกับถ้วยยุโรป) |

## 2. รูปแบบฟอร์มที่แฟนบอลไทยคุ้น

- **ฟอร์มไหล**: ชนะต่อเนื่อง ≥3 นัด แบบยิงนัดละ ≥2 ลูก → ใช้คำว่า "เครื่องร้อน" / "ไหลไม่หยุด"
- **ฟอร์มแกว่ง**: ผล WWLLW หรือสลับทุกนัด → ใช้คำ "ทรงไม่นิ่ง" / "ลุ้นเหนื่อย"
- **ฟอร์มเหย้า/เยือนแตกต่าง**: เหย้าได้ ≥2.2 แต้ม/นัด แต่เยือน ≤0.8 → ชี้ให้คนแทงรู้ว่าต้องดูสนามด้วย
- **ฟอร์มหลังพักทีมชาติ**: ทีมใหญ่กลับมาโดยขุมกำลังไม่สมบูรณ์ ให้พูดถึง "อยู่ระหว่างปรับจูน" หรือ "กำลังเรียกจังหวะ"

## 3. วิธีเขียนย่อหน้าฟอร์ม

1. **กำหนดโทน**: ใช้ xG diff + W/D/L เพื่อบอกว่า "กำลังร้อน" หรือ "แผ่ว"
2. **เสริมตัวเลขไทยเข้าใจง่าย**: เช่น "5 นัดซัด 11 ลูก เสียแค่ 3" แทนการพูด xG ตรงๆ
3. **โยงสรีระ/สภาพจิตใจ**: เช่น "เพิ่งบุกชนะคู่แค้น ทำให้อารมณ์ทีมกำลังดี"
4. **ปิดด้วยเดิมพัน**: "โมเมนตัมยังเทโอเวอร์ 2.5" หรือ "ทรงนี้ต่อ -0.5 ยังพอไหว"

## 4. Shortcut Metrics

- **xG Swing (XS)** = (xG For last 3) - (xG Against last 3)
- **Tempo Index (TI)** = (Shots For + Shots Against)/match → TI>28 = เกมเปิด, <20 = เกมปิด
- **Press Heat (PH)** = Tackles + Interceptions ในแดนคู่แข่ง → PH>15 = เพรสสูง, ใช้กับทีมไทยที่ชอบบุก

## 5. ข้อควรระวังสำหรับลีกไทย

- ทีมไทยลีกบางสนามพื้นหญ้า/สภาพอากาศกระทบเกมเร็ว → ระบุถ้าเตะบ่ายหรือสนามฝนตกบ่อย (ราชมัง, ชลบุรี)
- ค่า xG จากแหล่งสากลอาจไม่มี → ใช้ตัวชี้วัดทดแทน: โอกาสยิงในเขตโทษ, จำนวนครั้งครอสสำเร็จ
- ฟอร์มทีมทหาร/มหาวิทยาลัยมักขึ้นกับนักเตะยืม → เช็คใบอนุญาตก่อนสรุป
