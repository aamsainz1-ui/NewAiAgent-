# Matchup & H2H Analysis Guide

## 1. ข้อมูลที่ต้องดึงทุกครั้ง

1. **Head-to-Head 5 นัด** — ดูทั้งผลและจำนวนสกอร์ (over/under) → ใช้คำ "ข่ม" หรือ "แพ้ทาง"
2. **สถานที่แข่ง** — เหย้า/เยือน, พื้นหญ้า, สนามกลาง, อุณหภูมิ/ความชื้น (สำคัญกับทีมยุโรปมาไทย)
3. **ช่วงเวลาเตะ** — คู่ดึกยุโรป vs คู่บ่ายไทย ส่งผลกับจังหวะ/ค่าน้ำ
4. **ผู้ตัดสิน** — ลีกไทย/เอเชียบางนัดมีผู้ตัดสินต่างชาติ → มีผลกับจำนวนใบเหลือง/จุดโทษ
5. **เดิมพันจากตลาด** — ราคาเปิด vs ปัจจุบัน (ไหลขึ้น/ลง) → บอก narrative "ราคาต่อไหลขึ้น" = นักเล่นเทฝั่งต่อ

## 2. วิธีสรุป H2H

```
เจอกัน 5 ครั้งหลังสุด {TEAM_A} ชนะ {A_WINS} ครั้ง ยิงรวม {A_GOALS} ลูก ถือว่าข่มอยู่พอสมควร เพราะ {TEAM_B} ยังเสียอย่างน้อย 1 ลูกทุกครั้ง แถมเล่นที่ {VENUE_TH} เจ้าบ้านยิ่งคุ้นสนาม
```

- ถ้าสกอร์ออกสูงซ้ำๆ → บอกว่า "คู่กันทีไรเปิดเกมแลก" → แนะนำ over
- ถ้าผลสูสีแต่ราคาต่อแรง → ย้ำว่า "ต่อแพงต้องลุ้นยิงขาดครั้งแรกในรอบ {YEARS} ปี"

## 3. เน้น Match-up รายจุด

| ประเด็น | ตัวอย่างประโยค |
| --- | --- |
| ปีก vs แบ็ค | "ปีกซ้ายลิเวอร์พูลเลี้ยงกินตัว 9 ครั้งต่อเกม ต้องลุ้นว่าแบ็คขวาเชลซีรับอยู่หรือไม่" |
| กองกลาง | "แดนกลางเจ้าบ้านเน้นโฮลดิ้งดูโอ ทำให้เจอทีมที่เล่นหมายเลข 10 จะเสียพื้นที่" |
| ลูกกลางอากาศ | "ทีมเยือนโดนลูกตั้งเตะ 7 ลูกใน 4 นัดหลัง เกมนี้เจ้าบ้านมีตัวสูงสองคน" |

## 4. สภาวะตลาดไทย

- ถ้าคนไทยนิยมสเต็ปคู่นี้ ให้บอก "เป็นคู่ล็อกสเต็ป" → กระตุ้นให้ระวังการพลิก
- กลุ่มนำเล่นชอบเล่นราคาน้ำแดง → ให้ข้อมูลว่าค่าน้ำไหลลงทิศไหน

## 5. สรุปท้ายบท

ปิดด้วย **ข้อสรุปเดิมพัน** เช่น:
- "ทรงนี้รองลูกครึ่งดีกว่า"
- "สถิติยิงกันหนัก กดสูง 2.5"
- "ถ้าราคาเปิดกลับมาครึ่งควบ ให้รอไลฟ์"
