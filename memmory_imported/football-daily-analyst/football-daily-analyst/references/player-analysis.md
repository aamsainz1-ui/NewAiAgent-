# Player Analysis Cheat Sheet (Thai Bettor Focus)

## Metrics ที่ควรดึง (ต่อ 90 นาที)

| Metric | ใช้กับ | วิธีตีความแบบไทย |
| --- | --- | --- |
| Goals + Assists / 90 | ตัวรุก | >0.60 = "ยืนค้ำสกอร์" / <0.30 = ฟอร์มฝืด |
| Shot Quality (xG/Shot) | กองหน้า | >0.18 = ได้ยิงในกรอบเยอะ (หาคำว่า "มีพิษในเขตโทษ") |
| Progressive Carries | ปีก/กองกลาง | ≥7 ครั้ง/90 = "พาบอลกินตัว" |
| Key Passes | เพลย์เมกเกอร์ | ≥2.0/90 = "จ่ายเปิดช่อง" |
| Defensive Actions (Tackles+Interceptions) | กองกลางรับ/แบ็ค | ≥6/90 = เพรสเกาะติด |
| Duel Win % | เกมแผงหลัง | >60% = "ปะทะดีกว่า" |
| Pressures in Final Third | หน้าเป้า/ปีก | ≥6/90 = เพรสสูงจัด |
| Expected Threat (xT) Contribution | ปีก/แบ็คบุก | ใช้คำว่า "สร้างอันตรายในพื้นที่สุดท้าย" |

## วิธีเล่าให้คนไทยเข้าใจ

1. **เริ่มด้วยบทบาท** — "ดาวยิงเบอร์ 9", "ปีกซ้ายสปีดจัด", "ตัวตัดเกม"
2. **เชื่อมกับสถิติ** — "ยิง 4 ลูก + 2 แอสซิสต์ จาก 5 นัดหลัง (0.85 ต่อ 90)"
3. **พูดถึงความสม่ำเสมอ** — "สร้างโอกาส ≥2 ครั้งทุกนัด"
4. **สรุปผลต่อแผนพนัน** — "มีลุ้น Anytime Scorer" หรือ "ฝั่งต่อคุมแดนกลาง"

## ตัวอย่างโครงประโยค

```
กองหน้าชาวบราซิลยิง 0.72 ประตู/90 จากค่า xG/shot สูงถึง 0.21 ชี้ว่ามีวินัยยืนในกรอบ ร่วมกับปีกซ้ายที่ลากบอลขึ้นหน้า 8 ครั้ง/เกม ทำให้ฝั่งเจ้าบ้านมีตัวอันตรายทั้งแทงทะลุและบุกริมเส้น
```

## แหล่งข้อมูลที่ดึงได้เร็ว

- football-data.org → `/v4/competitions/{code}/scorers` (Top scorers + goals/penalties)
- TheSportsDB → `lookupplayer.php?id=` (ข้อมูลตำแหน่ง/สโมสร)
- FBRef / StatsBomb (ถ้าเข้าถึง) สำหรับ per-90 detail → เก็บค่าที่ใช้บ่อยใส่ JSON ไว้ในโฟลเดอร์ `data/`

## รายการสั้นที่ควรดึงประจำ

- สภาพร่างกาย: มีอาการเจ็บ? พักเกมยุโรปหรือไม่
- วินัย: ใบเหลืองสะสม → ส่งผลต่อความดุดันเกมรับ
- สไตล์: เท้าถนัด, ถนัดเล่นสวนหรือครองเกม
- เคมี: สร้างโอกาสให้ใครบ่อยสุด (ดู assist map)

## Tips สำหรับลีกไทย

- นักเตะไทยมักเล่นหลายตำแหน่ง → ระบุว่าปรับเล่นวิงแบ็ค/ปีกในเกมล่าสุด
- ต่างชาติ 3+1: เช็คว่าโควต้าเอเชียลงหรือไม่ เพราะมีผลต่อจุดแข็งทีม
- สถิติในไทยลีก II ใช้เทียบตรงๆ ไม่ได้ ต้องพูดว่ามาจากระดับรอง
