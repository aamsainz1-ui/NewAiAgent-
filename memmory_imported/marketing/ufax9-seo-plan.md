## 5. Internal Linking Strategy

### หลักการ
- ทุกบทความใหม่ต้องลิงก์กลับมาหน้าหลัก (ufax9.com) อย่างน้อย 2 ลิงก์
- ทุกบทความต้องลิงก์ไปบทความที่เกี่ยวข้องอย่างน้อย 3 บทความ
- หน้าหลักต้องมีลิงก์ไปบทความอย่างน้อย 10 ลิงก์ (เป้าหมาย 120+ links)

### แผนผัง Internal Link (Hub & Spoke Model)

```
                    ┌─────────────────┐
                    │   หน้าหลัก       │
                    │   ufax9.com      │
                    └────────┬────────┘
                             │
         ┌───────────────────┼───────────────────┐
         │                   │                   │
    ┌────▼────┐        ┌────▼────┐        ┌────▼────┐
    │ แทงบอล  │        │ คาสิโน  │        │ สล็อต   │
    │  Hub    │        │  Hub    │        │  Hub    │
    └────┬────┘        └────┬────┘        └────┬────┘
         │                   │                   │
    ┌────▼────┐        ┌────▼────┐        ┌────▼────┐
    │บทความ 1 │        │บทความ 7 │        │บทความ 8 │
    │บทความ 6 │        │บทความ 17│        │บทความ 18│
    │บทความ 14│        │บทความ 19│        │บทความ 10│
    │บทความ 15│        └─────────┘        └─────────┘
    │บทความ 20│
    └─────────┘
```

### Anchor Text Strategy
- ใช้ anchor text ที่หลากหลาย ไม่ซ้ำกัน
- ผสมระหว่าง exact match, partial match, branded, และ generic
- ตัวอย่าง:
  - Exact: "แทงบอลออนไลน์ UFAX9"
  - Partial: "เว็บแทงบอลที่ดี"
  - Branded: "UFAX9"
  - Generic: "อ่านเพิ่มเติม", "คลิกที่นี่", "ดูรายละเอียด"
- สัดส่วน: Exact 30% / Partial 30% / Branded 20% / Generic 20%

### ลิงก์ที่ต้องเพิ่มในหน้าหลักทันที
1. ลิงก์ไปหน้าสมัครสมาชิกในทุก H2 section (อย่างน้อย 8 ลิงก์)
2. ลิงก์ไปหน้าโปรโมชั่น 3-5 ลิงก์
3. ลิงก์ไปบทความ/คู่มือ 5-8 ลิงก์
4. ลิงก์ไป FAQ 2-3 ลิงก์
5. Footer links ไปทุกหมวดหลัก 10+ ลิงก์

## 8. หมายเหตุเพิ่มเติม

### On-Page SEO Checklist สำหรับทุกบทความ
- [ ] Title tag มี keyword หลัก อยู่ใน 60 ตัวอักษร
- [ ] Meta description มี keyword + CTA อยู่ใน 160 ตัวอักษร
- [ ] URL สั้น มี keyword ภาษาอังกฤษ เช่น /blog/web-betting-2025
- [ ] H1 มี 1 อันเท่านั้น = Title ของบทความ
- [ ] H2 อย่างน้อย 5 หัวข้อ
- [ ] H3 อย่างน้อย 3 หัวข้อ
- [ ] รูปภาพมี alt text ที่มี keyword
- [ ] Internal links อย่างน้อย 5 ลิงก์
- [ ] External links อย่างน้อย 1 ลิงก์ (ไปแหล่งข้อมูลน่าเชื่อถือ)
- [ ] Article Schema markup
- [ ] Breadcrumb Schema markup
- [ ] Mobile-friendly ตรวจสอบแล้ว
- [ ] Page speed ≤ 3 วินาที

### เครื่องมือที่แนะนำ
- **Google Search Console** — ติดตาม indexing และ ranking
- **Google Analytics 4** — วัด traffic และ user behavior
- **Ahrefs / SEMrush** — ติดตาม keyword ranking
- **PageSpeed Insights** — ตรวจสอบความเร็วเว็บ
- **Schema Markup Validator** — ตรวจสอบ structured data
