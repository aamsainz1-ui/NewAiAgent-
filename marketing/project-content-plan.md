#  Marketing Content Plan

## เป้าหมาย
- 8,000 ยูสฝาก/เดือน
- งบรวม ฿2,000,000
- CPA ≤ ฿250

---

## ทีม 1: TikTok Ads + Organic — เก่ง + พิม

### หน้าที่
- ถ่ายคลิปสั้น TikTok 3-5 คลิป/วัน
- ยิง TikTok Ads (Spark Ads + In-Feed)
- โพสต์ organic TikTok ทุกวัน
- ทำ A/B Test hook แรก 3 วินาที

### Content Ideas
1. **รีวิวชนะ** — โชว์หน้าจอถอนเงินสำเร็จ (เบลอข้อมูลส่วนตัว)
2. **สอนเล่น** — วิธีสมัคร, วิธีฝาก-ถอน, วิธีเล่นสล็อต/บอล
3. **โปรโมชั่น** — สมัครใหม่รับโบนัส, คืนยอดเสีย
4. **Trend Hijack** — ใช้เสียง/เทรนด์ TikTok มาทำคอนเทนต์
5. **Behind the scenes** — ทีมงาน, บรรยากาศออฟฟิศ

### KPI
- Reach: 500K/สัปดาห์
- คลิก Link: 2,000/วัน
- ยูสสมัครจาก TikTok: 3,000/เดือน
- งบ Ads: ฿800,000/เดือน

### ตารางโพสต์
| เวลา | ประเภท |
|-------|--------|
| 10:00 | สอนเล่น / รีวิว |
| 14:00 | โปรโมชั่น |
| 18:00 | Trend / เอนเตอร์เทน |
| 21:00 | รีวิวชนะ / ผลบอล |

## ทีม 2: AI Video Clips → TikTok + FB Ads — แบงค์ + เม่า

### หน้าที่
- สร้างคลิป AI (ใช้ Runway, Pika, HeyGen, CapCut)
- ยิง Ads ทั้ง TikTok + Facebook
- ทำ Creative Variations 5-10 ชิ้น/สัปดาห์
- Track performance แต่ละ creative

### Content Ideas
1. **AI Avatar** — ตัวละคร AI พูดแนะนำเว็บ, โปรโมชั่น
2. **Motion Graphics** — กราฟิกเคลื่อนไหวโชว์โปร, odds, ผลบอล
3. **Testimonial AI** — รีวิวจากยูสเซอร์ (AI voice + ภาพ)
4. **Countdown/Urgency** — โปรหมดเขต, แมตช์คืนนี้
5. **Comparison** — เปรียบเทียบ  กับเว็บอื่น (ฝาก-ถอนเร็วกว่า)

### KPI
- Creative output: 10 ชิ้น/สัปดาห์
- CTR: ≥ 2%
- CPA จาก FB: ≤ ฿200
- CPA จาก TikTok: ≤ ฿250
- ยูสสมัครจาก Ads: 3,000/เดือน
- งบ FB: ฿600,000 | งบ TikTok: ฿200,000

### Ad Structure
- Campaign: Conversions (สมัคร/ฝากเงิน)
- Ad Set: แยกตาม Audience (บอล/สล็อต/คาสิโน)
- Creative: 3-5 variations ต่อ ad set
- Optimize: ปิด creative ที่ CPA > ฿300 ภายใน 48 ชม.

## ทีม 3: กลุ่มนำเล่น + SMS — ลัน + โอ๊ต + บอม

### หน้าที่
- ดูแลกลุ่มนำเล่น LINE/Telegram (โพสต์ทิป, ผลบอล, โปร)
- ส่ง SMS campaign แบ่ง segment
- ติดตามผล conversion จาก SMS
- Re-engage ยูสเก่าที่ไม่ได้เล่นนาน

### SMS Templates

#### 1. ต้อนรับยูสใหม่
```
🎉 ยินดีต้อนรับสู่ !
สมัครวันนี้รับโบนัส 100%
ฝากขั้นต่ำ 100 บาท
👉 .com
ถอนได้ไม่อั้น 24 ชม.
```

#### 2. โปรโมชั่นประจำสัปดาห์
```
🔥 โปรพิเศษสัปดาห์นี้!
ฝาก 500 รับ 1,000
เล่นได้ทุกเกม สล็อต บอล คาสิโน
👉 .com
จำกัด 100 สิทธิ์แรกเท่านั้น!
```

#### 3. แมตช์คืนนี้ (บอล)
```
⚽ คืนนี้ห้ามพลาด!
[ทีม A] vs [ทีม B] เวลา [XX:XX]
วิเคราะห์: [ทีม] ได้เปรียบ
แทงบอลออนไลน์ค่าน้ำดีที่สุด
👉 .com
```

#### 4. Re-engage ยูสเก่า
```
😊 คิดถึงคุณนะ! กลับมาเล่นกับเรา
รับโบนัสคืนยอดเสีย 10%
ไม่ต้องทำเทิร์น ถอนได้เลย
👉 .com
```

#### 5. ผลชนะ / Jackpot
```
🏆 ยินดีด้วย! ยูสเซอร์  ถอน [X] บาท!
คุณก็ทำได้ ลองเลยวันนี้
สมัครฟรี ฝาก-ถอนออโต้
👉 .com
```

### กลุ่มนำเล่น Content Plan
| เวลา | ประเภท |
|-------|--------|
| 09:00 | สวัสดีตอนเช้า + โปรวันนี้ |
| 12:00 | วิเคราะห์บอลบ่าย/ค่ำ |
| 17:00 | ทิปบอลค่ำ + odds |
| 20:00 | Live วิเคราะห์แมตช์ |
| 22:00 | สรุปผล + โปรพรุ่งนี้ |

### SMS Segmentation
| Segment | เงื่อนไข | ข้อความ | ความถี่ |
|---------|----------|---------|---------|
| ยูสใหม่ | สมัคร < 7 วัน | ต้อนรับ + โบนัส | 1 ครั้ง |
| Active | ฝากใน 7 วัน | โปรพิเศษ + แมตช์ | 2-3/สัปดาห์ |
| Dormant | ไม่ฝาก > 14 วัน | Re-engage + คืนยอดเสีย | 1/สัปดาห์ |
| VIP | ฝากรวม > 50K | โปร Exclusive | 1/สัปดาห์ |
| Churned | ไม่ฝาก > 30 วัน | โบนัสพิเศษกลับมา | 2/เดือน |

### KPI
- กลุ่มนำเล่น: สมาชิก 5,000+/กลุ่ม
- SMS Conversion Rate: ≥ 3%
- Re-engage Rate: ≥ 15%
- ยูสกลับมาฝากจาก SMS: 2,000/เดือน
- งบ SMS: ฿200,000 | งบอื่น: ฿200,000
